Github国内访问慢？

2021-07-21 0 评论 字数统计: 262(字) 阅读时长: 1(分)

修改 HOST

由于网络代理商的原因，github 的 CDN 被某墙屏了，所以访问下载很慢。咱们可在本地直接绑定 host，绕过 dns 解析，该方法也可加速其他因为 CDN 被屏蔽导致访问慢的网站。

查询网址 IP

  • 访问https://www.ipaddress.com，分别输入 github.com 和 github.global.ssl.fastly.net，查询 IP 地址。

修改 hosts 文件

  • windows 系统的 hosts 文件的位置如下：C:\Windows\System32\drivers\etc\hosts
  • mac/linux 系统的 hosts 文件的位置如下：/etc/hosts

更新本地 DNS 缓存

  • windows
  • linux（unbantu 为例）
  • mac

使用镜像网站

常用镜像网站

https://hub.fastgit.org

使用

例如要访问项目https://github.com/jaywcjlove/linux-command

只需将路径中的 https://github.com 修改为 https://hub.fastgit.org 即可
