前提

  • github账号
  • node.js
  • git

创建Git仓库

打开github，新建仓库。

仓库名称命名格式：用户名.github.io。例如，我的用户名为g-ydg，则对应为g-ydg.github.io。

创建完成后，点击设置。

点击Pages选项卡，如下图所示即可。

打开浏览器访问g-ydg.github.io，若正常访问，则设置正常。

Hexo项目搭建

新建文件夹，点击文件夹右键打开 Git Bush Here。
输入命令安装hexo。


安装完成后，进行hexo初始化。


输入 hexo g，进行静态部署。

部署完成后，输入 hexo s 启动服务。

打开浏览器，访问启动地址。

部署到github

打开项目目录下的_config.yml文，滑动到文件底部，修改deoloy配置。
